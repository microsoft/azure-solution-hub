개인정보 및 분석 고지 (Privacy Notice)

본 웹사이트는 방문자가 사이트를 어떻게 이용하는지 이해하고 콘텐츠와 사용자 경험을 개선하기 위해 Google Analytics와 Microsoft Clarity를 사용합니다.

수집될 수 있는 정보는 다음과 같습니다.

  • 페이지 조회수 (Page views)
  • 유입 경로 (Referring sources)
  • 브라우저 및 기기 정보 (Browser and device information)
  • 익명 사용 통계 (Anonymous usage statistics)

개인을 식별할 수 있는 정보(PII)는 의도적으로 수집하지 않습니다. 본 사이트를 계속 이용하시는 경우 위에 설명된 분석 기술의 사용에 동의하는 것으로 간주됩니다.

English

This website uses Google Analytics and Microsoft Clarity to understand how visitors use the site and improve the content and user experience.

The collected information may include:

  • Page views
  • Referring sources
  • Browser and device information
  • Anonymous usage statistics

No personally identifiable information is intentionally collected. By continuing to use this site, you agree to the use of analytics technologies as described above.
