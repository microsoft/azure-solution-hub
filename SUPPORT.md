# Support

## 이슈 등록 및 도움 요청

이 프로젝트는 버그와 기능 요청을 추적하기 위해 **GitHub Issues** 를 사용합니다. 새 이슈를 등록하기 전에 중복을 피하기 위해 [기존 이슈](https://github.com/Anna-Jeong-MS/azure-solution-hub/issues)를 먼저 검색해 주세요. 새로운 문제는 새 Issue로 등록해 주시기 바랍니다.

- **버그 신고 / 기능 요청**: [GitHub Issues](https://github.com/Anna-Jeong-MS/azure-solution-hub/issues) 에 등록
- **콘텐츠(워크샵·솔루션) 추가 제안**: Issue 또는 Pull Request 로 제안 (작성 방법은 [README](README.md) 참고)
- **일반 문의**: 위 GitHub Issues 채널을 통해 문의해 주세요.

## 지원 정책

이 프로젝트에 대한 지원은 위에 안내된 커뮤니티 리소스(GitHub Issues)로 한정됩니다. 응답은 유지 관리자의 가용성에 따라 제공되며, 정해진 응답 시간(SLA)은 보장되지 않습니다.
